# 📂 Clasificador PDF

**AutoRename OCR** es una herramienta potente y sencilla para automatizar el renombrado de archivos PDF mediante el reconocimiento óptico de caracteres (OCR). Ideal para clasificar facturas, matrículas o documentos académicos de forma masiva.

## ✨ Características

- **Extracción Inteligente:** Busca patrones de texto (inicio, fin o contenido) y úsalos como variables.
- **Clasificación Automática:** Identifica tipos de documentos por palabras clave.
- **Constructor de Nombres:** Interfaz visual tipo "rompecabezas" para definir el formato final.
- **Portable:** No requiere instalación de Python ni Tesseract (en su versión de lanzamiento).

## 🚀 Instalación (Windows)

1. Ve a la sección de [Releases](https://github.com/AaronHero03/PDF_Classifier/releases).
2. Descarga `setup.exe`.
3. Ejecuta el instalador y sigue las instrucciones.

## 🛠️ Desarrollo (Para programadores)

Si quieres ejecutar el código fuente:

1. Clona el repositorio.
2. Crea un entorno virtual: `python -m venv venv`.
3. Instala dependencias: `pip install -r requirements.txt`.
4. Asegúrate de tener Tesseract en la carpeta `tesseract_bin/`.
5. Ejecuta: `python main.py`.
